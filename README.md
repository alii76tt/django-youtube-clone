```
pip install -r requirements.txt
```

## Installation
```sh
$ git clone https://github.com/alii76tt/django-youtube-clone
$ cd django-youtube-clone
$ pip install -r requirements.txt
$ python manage.py createsuperuser
$ python manage.py runserver
```
